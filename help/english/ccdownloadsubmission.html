Download Submissions

This is the command and control quick moderation area for download submissions. In this area you can edit, delete or approve download submissions. Select any combination of deletes or approves and click the submit button.

Field Description
Edit Click this to edit the download. Note that saving an edited download submission will approve it
Title Title of the file download
Version Version of the file download
Category The category to which the download was posted
